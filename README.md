This repository contains three simple, beginner-friendly C++ console applications.  
Each project demonstrates core programming concepts such as functions, conditionals, loops, arrays, and user interaction through the terminal.

### 1. Tic-Tac-Toe (XOX)
File: `Tic-Tac-Toe.cpp`

A classic Tic-Tac-Toe game where:
- The user plays as **X**
- The computer plays as **O**

- Clean console board display
- Turn-based gameplay

File: `Mini bank.cpp`

A small console-based banking simulation:
